01 / CONTEXT

WHOLE FILES, NOT HUNKS

Reviews load full changed files, tree-sitter semantic regions, and directly related code. Diff-only review is too shallow to trust.

02 / EVIDENCE

ANALYZER-BACKED FINDINGS

TypeScript, ESLint, Semgrep, secret and dependency scanners run first. The model triages signals instead of inventing them.

03 / GATEWAY

ROUTED, BUDGETED, AUDITED

A hosted gateway routes each review to the right model, enforces monthly budgets per provider, and writes an audit trail.

04 / GITHUB

COMMENTS ON THE RIGHT LINES

Webhook-driven reviews post one summary that updates in place, inline comments mapped to the diff, and suggestion blocks for fixes.
